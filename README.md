# Cambodian Financial Calculators

A comprehensive collection of financial calculators designed specifically for Cambodia, featuring tools for loans, mortgages, savings, and more. All calculations are performed client-side for privacy and speed.

## Features

- Loan Calculator
- Mortgage Calculator
- Savings Calculator
- Fixed Deposit Calculator
- Currency Converter
- Car Loan Calculator
- Monthly Budget Planner
- And many more financial tools

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

The app will be available at http://localhost:3000

## Build for Production

To create a production build:
```bash
npm run build
```

The built files will be in the `dist` directory.
